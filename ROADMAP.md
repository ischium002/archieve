# ROADMAP —— OC Archive 发展规划

2026-07-11 由 Fable 规划，坐骨拍板取舍。每条标了难度（★☆☆ 小 / ★★☆ 中 / ★★★ 大）。
用法：想做某条时，把该条目 + MAINTENANCE.md 一起喂给 agent 写 spec，简单条目 sonnet 级 agent 即可。
做完一条就把状态从 `[ ]` 改 `[x]`，砍掉的条目别删，标 `[-]` 留痕。

> 前提共识：本站是纯静态、无构建的档案馆气质站点。所有新功能顺着这个气质长，不引入构建工具、不加后端。

  - [x] h-paper 初版完成（2026-07-11，`ephemera/h-paper.html`），待坐骨改文案

  - [ ] 涂黑备忘录 spec 已写（2026-07-11，`specs/ephemera-redacted-memo.md`），待拍板后动工
